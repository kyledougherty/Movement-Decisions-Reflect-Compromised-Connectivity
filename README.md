# Movement-Decisions-Reflect-Compromised-Connectivity
Data and code necessary to reproduce results in the manuscript 'Movement Decisions Reflect Compromised Statewide Connectivity for Mountain Lions in California'

Data Access: the script 'Movement-Decisions Reflect-Compromised-Statewide-Connectivity.R' contains code on lines 7-12 to download input files from OneDrive.

'Movement-Decisions Reflect-Compromised-Statewide-Connectivity.R' also contain code to reproduce results reported in the paper.
